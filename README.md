# IS-23
Учебный репозиторий для студентов группы ИС-23
1. Добавление удаленного репозитория:
    git remote add <имя> <url>
    После, можно проверить список всех добавденных репозиториев командой git remote.
2. Подтянуть из добавленного репозитория все данные:
    git fetch origin - данная команда связывается с указанным удалённым проектом и забирает все те данные проекта, которых у вас ещё нет.
После изменения файлов проекта необходимо добавить эти файлы в индексацию, выполнив команду git add * - или слева через графический интерфейс, нажав "+" (stage changes), или отменить эти изменения, выполнив команду git reset или через Г\И, нажав на стрелочку (Discard Changes)
