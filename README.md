# IS-23
Учебный репозиторий для студентов группы ИС-23
1. После изменения файлов проекта необходимо добавить эти файлы в индексацию, выполнив команду git add * - или слева через графический интерфейс, нажав "+" (stage changes), или отменить эти изменения, выполнив команду git reset или через Г\И, нажав на стрелочку (Discard Changes)
2. После индексации нужно сформировать комит либо командной строкой либо через графический интерфейс. После выполнения коммита в терминал выводится отчёт о том, какие файлы были добавлены в репозиторий.
3. Пушим комммит (git push origin main), где origin - имя репозитория, а main - имя ветки
4. Подключить локальный репозиторий к удалённому git remote add origin (URL-репозитория), получить изменения из удалённого репозитория спомощью git pull origin, отправить локальные изменения в удалённый репозиторий спомощью git push origin main
